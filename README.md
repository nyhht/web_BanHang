# 🍳 MealKit - Hệ Thống Thương Mại Điện Tử Cung Cấp Thực Phẩm Sơ Chế Sẵn (Laravel 11)

MealKit là một nền tảng thương mại điện tử đột phá chuyên cung cấp các hộp nguyên liệu thực phẩm được sơ chế, định lượng sẵn kèm nước sốt độc quyền chuẩn vị, giúp người bận rộn hoàn thành bữa ăn ngon tại nhà chỉ trong 15 phút.

Dự án được xây dựng trên nền tảng **Laravel 11**, tích hợp hệ thống quản lý danh mục thực đơn thông minh, giỏ hàng, quy trình thanh toán qua PayPal, đánh giá sản phẩm, hệ thống gợi ý sản phẩm tự động và tối ưu hóa tìm kiếm bằng xử lý ngôn ngữ tự nhiên (NLP/NLTK).

---

## 🚀 Requirements
- **PHP**: `>= 8.2`
- **Composer**: `>= 2.x`
- **MySQL / MariaDB**
- **Python**: `>= 3.9` (Để vận hành API bổ trợ tìm kiếm NLP và hệ thống gợi ý sản phẩm)
---


## 📦 Features
📦 Quản lý Thực đơn & Danh mục Meal-kit: Phân loại món ăn thông minh theo chế độ (Gia đình, Healthy/Eat Clean, Món Chay, Lẩu/Nướng,...).

🛒 Giỏ hàng trực tuyến: Quản lý giỏ hàng linh hoạt dựa trên Session và lưu trữ Database.

💳 Thanh toán PayPal: Tích hợp cổng thanh toán quốc tế an toàn, xử lý quy trình bàn giao đơn hàng tự động.

🤖 AI Chatbot trợ lý bếp: Tích hợp Gemini API hỗ trợ tư vấn thực đơn và giải đáp công thức nấu ăn trực tiếp cho khách hàng.

📊 Tìm kiếm & Gợi ý nâng cao (NLP): Ứng dụng thư viện NLTK kết nối qua API cổng 5555 để chuẩn hóa từ khóa và tự động gợi ý món ăn liên quan dựa trên thói quen người dùng.

⭐ Tương tác người dùng: Hệ thống đánh giá xếp hạng (Reviews) sản phẩm và Danh mục yêu thích (Wishlist).

🖥️ Bảng điều khiển Admin Dashboard: Quản lý toàn diện đơn hàng, sản phẩm, doanh thu và người dùng hệ thống.
